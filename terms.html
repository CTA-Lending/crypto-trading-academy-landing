第一條 服務定義與性質

1.1 服務內容

本服務為一款自動化放貸軟體工具 (Automated Lending Software Tool)，透過 Telegram Bot 介面，協助用戶在 Bitfinex 交易所上自動執行融資放貸操作。功能包括但不限於：

  • 自動掛單放貸
  • 智能利率偵測與調整
  • 每日收益報告
  • 放貸參數自訂設定

1.2 服務性質聲明

本服務僅為 SaaS 軟體工具，絕非以下任何一種服務：

  • 非代操服務：本公司不代替用戶做任何投資決策
  • 非投資顧問：本公司不提供任何投資建議
  • 非代理交易：本公司不以用戶名義進行任何交易
  • 非資產託管：本公司不持有、不控制、不託管用戶的任何資產

用戶的資金始終存放在用戶自己的 Bitfinex 帳戶中，本公司無法提取、轉移或以任何方式動用用戶資金。

1.3 API 權限限制

本服務僅要求用戶提供 Funding（融資）權限 的 API Key。本公司明確禁止、也不需要用戶提供包含交易或提款權限的 API Key。

  1. 自行註冊 Bitfinex 帳戶並完成相關身份驗證（KYC）
  2. 自行建立具備 Funding 權限的 API Key
  3. 自行綁定 API Key 至本服務
  4. 自行設定放貸參數（最低利率、放貸天數、放貸金額等）
